
<h1> ServiceCreator </h1>

<h2>About: </h2>

**ServiceCreator** is a scripts for creating systemctl service from a Shell Script

<h3>How to use: </h3>
* Root user required

./ServiceCreator <Script Path>
example: ./ServiceCreator myscript.sh OR bash ServiceCreator <Script Path>
